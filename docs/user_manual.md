# User Manual

This is a work in progress, not much to see here yet. 

1. Choose a map from the dropdown menu
2. Select start (x,y) and goal (x,y) positions
3. Choose which pathfinding algorithm to run (A* vs JPS)
4. The shortest path will be displayed on the UI
5. Runtime, expanded nodes and path length are currently reported only in terminal - these will be added to UI next week. Visualization of the nodes expanded is also planned for the next week.
