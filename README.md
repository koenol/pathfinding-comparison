# pathfinding-comparison

This project was developed for the University of Helsinki’s TKT20010 – Algorithms and Artificial Intelligence Project course (Spring/2026). The goal is to compare the performance of JPS and A* algorithms for pathfinding on Warcraft 3 maps, provided by [movingai.com](https://movingai.com/benchmarks/wc3maps512/index.html).

# Installation Guide

Prerequisites:
- Poetry
- Python 3.13.x (>=3.13,<3.14)

Install dependendencies
```
poetry install
```
Run the app
```
poetry run invoke start
```

See [User Manual](docs/user_manual.md) for more commands
